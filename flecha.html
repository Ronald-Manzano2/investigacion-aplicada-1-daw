Funciones flecha

Estas funciones son una alternativa simplificada de declarar los métodos o funciones que normalmente usamos; no obstante, tiene una utilidad limitada.

Se caracteriza por:

Se caracterizan por la utilización del símbolo "Flecha" ( => )

¿Cómo funciona?

Esta función evalúa los parámetros a la izquierda de => y devuelve la expresión a su derecha.

Principales limitaciones

Esta función, como se mencionó anteriormente tiene ciertas limitaciones, las cuales pueden poner en aprietos si no lo tomamos en cuenta.

No se puede usar como un Método.

No tiene enlaces o referencias a this o super.

No contiene arguments, ni palabras clave new.target

No está disponible para métodos call, apply y bind que necesitan una referencia establecida.

Tampoco puede funcionar o ser utilizado como un constructor

Ni mucho menos con yield, usado para pausar y reanudar funciones generadoras.
