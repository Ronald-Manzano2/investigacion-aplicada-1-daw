Operador Spread (...)

Descripción

El operador extendido o spread, sirve para acortar la sintaxis de listas de parámetros en algunas estructuras. El principal uso es en:

Funciones

myFunction(...iterableObj);

Arreglos

[...iterableObj, '4', 'five', 6];

Objetos

let objClone = { ...obj };

Sintaxis

Como en los ejemplos anteriores se demuestra, la sintaxis de este operador suele se variada, dependiendo de la estructura donde se use.
